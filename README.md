📌 Project Overview
This capstone project analyzes Walmart's historical weekly sales data across 45 stores to uncover patterns in consumer behavior, seasonal trends, and the impact of external factors like fuel price, CPI, temperature, and unemployment on retail sales performance.
The analysis was conducted using Tableau for data visualization and includes interactive dashboards, a Tableau workbook, and a written report summarizing key findings and strategic recommendations.

📊 Dataset Features
Column	Description
Store	Store number (1–45)
Date	Week of sales
Weekly_Sales	Total sales for the week ($)
Holiday_Flag	Whether the week includes a public holiday (1 = Yes, 0 = No)
Temperature	Average regional temperature (°F)
Fuel_Price	Cost of fuel in the region
CPI	Consumer Price Index
Unemployment	Regional unemployment rate

🔍 Key Findings
•	Online and in-store sales spike significantly during holiday periods
•	Grocery and essential product categories consistently outperform others
•	Seasonal patterns in purchasing behavior call for dynamic inventory planning
•	Urban and rural stores exhibit distinct buying trends, suggesting region-specific marketing strategies
